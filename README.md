# 100 Days of Code 🐍💻

## About
This repository documents my journey through the **100 Days of Code: The Complete Python Pro Bootcamp** by Dr. Angela Yu, alongside web development projects. Each folder contains the code I wrote while learning and building projects from scratch — from beginner-level Python fundamentals to intermediate concepts like APIs, web scraping, automation, and GUI applications.
